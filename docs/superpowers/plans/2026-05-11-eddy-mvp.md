# eddy MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the eddy MVP — a mobile-portrait videosong creator where the layout-builder grid is the song. Each cell holds one recorded video+audio "voice"; pressing play composites all voices into one synced video+audio mix. Export deferred to v2.

**Architecture (after design + audit pass):**
- One WebGL canvas — the existing `webgl/renderer.ts` is extended to draw video-frame textures per leaf, falling back to leaf color when a cell has no clip / no frame yet.
- `LeafFrame` gains a stable `id` (the entity id), used as the compositor's keying.
- The playback render loop lives **inside** `canvas.tsx`'s existing `onSettled` block (not in a separate component), so it shares the renderer instance and coordinate space.
- During recording, the active cell renders the camera's live `HTMLVideoElement` (the MediaStream) as a texture source. Same `Map<id, source>` for clip frames and live preview.
- Audio: one `AudioContext` (single-resume on first user gesture), `AudioBufferSourceNode` per voice per play-pass, sample-accurate `start(when)`.
- Transport, anchor-take, clip store written fresh. Ports from `../eddy-1.0/` for the WebCodecs/mediabunny mechanics where they're battle-tested.

**Tech Stack:** Solid 2.x (`solid-js`, `@solidjs/signals`, `@solidjs/web`), Vite, TypeScript, `@bigmistqke/view.gl` (WebGL), `mediabunny` (mux/demux), WebCodecs (`VideoEncoder`, `VideoDecoder`, `AudioEncoder`, `AudioDecoder`), Web Audio, `getUserMedia`, `MediaRecorder`, Playwright.

**Reference:** Design spec at `docs/superpowers/specs/2026-05-11-eddy-mvp-design.md` is authoritative — defer to it on any scope conflict.

**Three phases:**
- **Phase 0 — Setup.** Deps, fixtures, Entity.id, LeafFrame.id, tool-gating fix, audio-context module, typed window augments, fake-media config.
- **Phase 1 (M1) — Single-cell record + playback with live preview.** Ports, fresh `frameAt` ring, demuxer spike, clip store, transport, renderer extension, render-loop integration, HUD wiring.
- **Phase 2 (M2) — Multi-cell synced playback.** Generalise transport + render loop to N clips.
- **Phase 3 (M3) — Anchor-take semantics.** First clip sets length, subsequent clamped, monitor during record, delete/reset.

## Conventions (apply to every task)

These are the user's standing preferences. The implementer MUST follow them:

- **Brace single-statement `if` / early-return blocks**, body on its own line.
- **No abbreviated or single-char names.** `index` not `i`, `channel` not `ch`, `element` not `el`. Setter callbacks echo the underlying name: `setApp(app => ...)`. Aliased accessors get an underscore prefix when shadowing: `const _viewport = context.viewport`.
- **Typed window augments** via `src/global.d.ts`. No `(window as any)`. Reads/writes go through the typed interface.
- **`logAction(type, payload)`** for every user-driven input (per `src/utils.ts`). This is what `tests/helpers.ts:runActions` replays. New action types must be added to the `Action` union in `tests/helpers.ts`.
- **pnpm**, not npm/npx. `pnpm add`, `pnpm exec`, `pnpm run`.
- **Bash timeouts:** keep small (≤ 30s); for long commands use `run_in_background`.

### Solid 2.x rules (project is on `solid-js` 2.0.0-beta.9)

This project uses Solid 2.x beta APIs. They differ meaningfully from 1.x. The implementer MUST follow these:

- **Effects are split: compute → apply.** `createEffect(compute, effect)` two-arg form only. The compute reads reactive values; the apply runs side effects and may return a cleanup. The single-arg form is gone.
- **Mount + cleanup → `onSettled(() => { setup; return cleanup })`.** `onMount` is gone.
- **`onSettled` and `effect` callbacks are unowned.** Default approach: don't nest reactive primitives (`createSignal`, `createEffect`, `createMemo`, `createStore`, `createProjection`) inside them. Define primitives at component scope; communicate with `onSettled`-bound code via plain refs (`let drive = null`-style) — see `src/components/canvas.tsx:60-62` and `:256-265` for the canonical pattern. **Escape hatch:** when you genuinely need a reactive primitive in an unowned scope, wrap the creation in `runWithOwner(owner, () => …)` with a captured owner. Prefer the plain-ref pattern unless you have a real reason.
- **No writes under owned scope.** Writing to a signal/store inside `createEffect`'s *compute* or inside a `createMemo` throws in dev. Writes belong in: event handlers, `onSettled`, the *apply* phase of `createEffect`, and `untrack(...)` blocks. `ownedWrite: true` is for internal signals written from their own owner — never as an escape hatch for app state.
- **`createStore`'s setter is draft-first (produce-style by default).** Do NOT import `produce`. Form: `setStore(draft => { draft.x = y })`. Returning a value from the setter performs a shallow replacement.
- **Imports moved.** `createStore`, `reconcile`, `snapshot`, `storePath` come from `solid-js` (not `solid-js/store`). DOM runtime is `@solidjs/web` (not `solid-js/web`).
- **Strict top-level reactive reads.** Reading a signal/store at the top level of a component body warns in dev. Reads belong inside JSX expressions, inside `createMemo`/`createEffect`, or wrapped in `untrack`. This also applies to control-flow function children (`<Show>{u => …}</Show>`): the callback body is structure-building — wrap the work in JSX expressions or `untrack`.
- **`Show`/`Switch`/`Match` function children receive accessors.** `<Show when={user()}>{u => <Profile user={u()} />}</Show>` — note `u()`, not `u`.
- **Context: `<Context value={state}>`**, not `<Context.Provider>`. `useContext` on a no-default context throws at runtime if no provider — no need for `T | undefined` guards. Use a non-null assertion when consuming: `const context = useContext(Context)!` (matches the existing `canvas.tsx:46` pattern).
- **Stores in effect *compute* phases:** if you read multiple nested properties for serialization-style work, use `deep(store)` so the reads happen in the tracked compute phase. Reading deeply from the *apply* phase warns (STRICT_READ_UNTRACKED). The MVP doesn't currently need this, but flag if you find yourself in that shape.
- **No `batch` — use `flush()`** if you need synchronous read-after-set in tests/imperative code. Setters are batched on microtasks by default.

```typescript
function onWrapperClick(event: MouseEvent) {
  // Cell selection works in any tool mode — handles only render
  // when a tool is active (see canvas.tsx:275), but the underlying
  // selection signal drives transport + recording too.
  const wrapperRect = wrapperElement.getBoundingClientRect()
  const screenX = event.clientX - wrapperRect.left
  const screenY = event.clientY - wrapperRect.top
  const canvasX = screenX - lastViewport.x
  const canvasY = screenY - lastViewport.y
  for (const leaf of lastLeaves) {
    if (
      canvasX >= leaf.rect.x &&
      canvasX < leaf.rect.x + leaf.rect.width &&
      canvasY >= leaf.rect.y &&
      canvasY < leaf.rect.y + leaf.rect.height
    ) {
      logAction("tap-frame", { path: leaf.path })
      context.setSelection({ path: leaf.path, depth: 0 })
      return
    }
  }
}
```

### Task 1.5: Demuxer wrapper

**Files:** `src/media/demuxer.ts`

We don't port `../eddy-1.0/packages/media/src/make-demuxer.ts` verbatim — too much over-engineering for the MVP. Instead, write a small wrapper informed by the Task 1.4 spike.

- [ ] **Step 1: Write the wrapper**

Create `src/media/demuxer.ts`:

```typescript
import { Input, BlobSource } from "mediabunny"

export type DemuxResult = {
  audio: {
    chunks: EncodedAudioChunk[]
    config: AudioDecoderConfig
  }
  video: {
    chunks: EncodedVideoChunk[]
    config: VideoDecoderConfig
    durationSeconds: number
  }
}

/**
 * Demux a recorded Blob into encoded audio + video chunks plus their
 * decoder configs. Adjust this against the actual mediabunny 1.27 API
 * surface — the spike in Task 1.4 produced the notes for filling in
 * the iteration / config retrieval. Throw with a clear message if a
 * track is missing.
 */
export async function demuxBlob(blob: Blob): Promise<DemuxResult> {
  const input = new Input({ source: new BlobSource(blob) })

  // ↓ Names below are tentative — verify against mediabunny .d.ts. ↓
  const audioTrack = await input.getPrimaryAudioTrack()
  const videoTrack = await input.getPrimaryVideoTrack()
  if (audioTrack === null) {
    throw new Error("demuxBlob: blob has no audio track")
  }
  if (videoTrack === null) {
    throw new Error("demuxBlob: blob has no video track")
  }

  const audioChunks: EncodedAudioChunk[] = []
  for await (const packet of audioTrack.packets()) {
    audioChunks.push(new EncodedAudioChunk({
      type: packet.type === "key" ? "key" : "delta",
```

```typescript
/**
 * Decode all chunks into a single AudioBuffer.
 * For MVP: short clips (seconds to ≤ ~5 minutes), held in memory.
 */
export async function decodeToAudioBuffer(
  chunks: EncodedAudioChunk[],
  config: AudioDecoderConfig,
): Promise<AudioBuffer> {
  const decoded: AudioData[] = []

  const decoder = new AudioDecoder({
    output: (data) => {
      decoded.push(data)
    },
    error: (error) => {
      throw error
    },
  })
  decoder.configure(config)

  for (const chunk of chunks) {
    decoder.decode(chunk)
  }
  await decoder.flush()
  decoder.close()

  if (decoded.length === 0) {
    throw new Error("decodeToAudioBuffer: decoder produced no AudioData")
  }

  const sampleRate = decoded[0].sampleRate
  const numberOfChannels = decoded[0].numberOfChannels
  const totalFrames = decoded.reduce((sum, data) => sum + data.numberOfFrames, 0)

  const buffer = audioContext().createBuffer(numberOfChannels, totalFrames, sampleRate)

  let writeOffset = 0
  for (const data of decoded) {
    for (let channel = 0; channel < numberOfChannels; channel++) {
      const destination = buffer.getChannelData(channel).subarray(writeOffset)
      data.copyTo(destination, { planeIndex: channel, format: "f32-planar" })
    }
    writeOffset += data.numberOfFrames
    data.close()
  }

  return buffer
}
```

```typescript
/**
 * Long-lived per-clip video decoder with a small in-memory frame
 * ring. `frameAt(t)` returns the frame whose [pts, pts+duration)
 * covers `t` (microseconds), or the most recent frame older than
 * `t` if exact coverage isn't decoded yet (better than rendering
 * black during a brief decode hiccup). Older frames are closed and
 * dropped on a sliding window.
 */

const MAX_FRAMES_BEHIND = 4
const MAX_FRAMES_AHEAD = 24

export type VideoSource = {
  /** Make sure frames covering tMicros are queued for decode. */
  prime(tMicros: number): void
  /**
   * Return the frame whose pts interval covers tMicros, or the
   * closest earlier decoded frame if exact coverage isn't ready.
   * Returns null only when nothing has decoded yet.
   */
  frameAt(tMicros: number): VideoFrame | null
  /** Reset for a fresh loop pass. Frees buffered frames. */
  reset(): void
  close(): void
}

type RingEntry = {
  frame: VideoFrame
  start: number
  end: number
}

export function makeVideoSource(
  chunks: EncodedVideoChunk[],
  config: VideoDecoderConfig,
): VideoSource {
  const ring: RingEntry[] = []
  let nextChunkIndex = 0
  let decoderError: Error | null = null

  const decoder = new VideoDecoder({
    output: (frame) => {
      const start = frame.timestamp
      const end = start + (frame.duration ?? 0)
      ring.push({ frame, start, end })
      ring.sort((a, b) => a.start - b.start)
    },
    error: (error) => {
      decoderError = error
    },
  })
  decoder.configure(config)

  function prime(tMicros: number) {
    if (decoderError !== null) {
      throw decoderError
    }
    // Feed forward until decoder queue is full enough to cover ahead.
    while (
      nextChunkIndex < chunks.length &&
      decoder.decodeQueueSize < MAX_FRAMES_AHEAD
    ) {
      decoder.decode(chunks[nextChunkIndex])
      nextChunkIndex++
    }
    // Evict frames behind window.
    const cutoff = tMicros - MAX_FRAMES_BEHIND * 33_333 // ~33ms/frame margin
    while (ring.length > 0 && ring[0].end < cutoff) {
      ring[0].frame.close()
      ring.shift()
    }
  }

  function frameAt(tMicros: number): VideoFrame | null {
    if (ring.length === 0) {
      return null
    }
    // Exact coverage.
    for (const entry of ring) {
      if (tMicros >= entry.start && tMicros < entry.end) {
        return entry.frame
      }
    }
    // Closest earlier (decoder is behind real time).
    let candidate: VideoFrame | null = null
    for (const entry of ring) {
      if (entry.start <= tMicros) {
        candidate = entry.frame
      } else {
        break
      }
    }
    return candidate
  }

  function reset() {
    for (const entry of ring) {
      entry.frame.close()
    }
    ring.length = 0
    nextChunkIndex = 0
    decoder.reset()
    decoder.configure(config)
  }

  function close() {
    for (const entry of ring) {
      entry.frame.close()
    }
    ring.length = 0
    decoder.close()
  }

  return { prime, frameAt, reset, close }
}
```

```typescript
export type CaptureHandle = {
  /** Underlying MediaStream — used by preview to attach to an HTMLVideoElement. */
  stream: MediaStream
  /** Stop recording and yield the encoded Blob. */
  stop(): Promise<Blob>
  /** Stop without keeping the blob (e.g. user cancelled). */
  cancel(): void
}

const PREFERRED_MIME_TYPES = [
  "video/webm;codecs=vp9,opus",
  "video/webm;codecs=vp8,opus",
  "video/webm",
]

function pickMimeType(): string {
  for (const candidate of PREFERRED_MIME_TYPES) {
    if (MediaRecorder.isTypeSupported(candidate)) {
      return candidate
    }
  }
  throw new Error("capture: no supported MediaRecorder mime type")
}

export async function startCapture(): Promise<CaptureHandle> {
  const stream = await navigator.mediaDevices.getUserMedia({ video: true, audio: true })
  const mimeType = pickMimeType()
  const recorder = new MediaRecorder(stream, { mimeType })
  const chunks: Blob[] = []
  recorder.ondataavailable = (event) => {
    if (event.data.size > 0) {
      chunks.push(event.data)
    }
  }
  recorder.start()

  function teardownStream() {
    for (const track of stream.getTracks()) {
      track.stop()
    }
  }

  function stop(): Promise<Blob> {
    return new Promise<Blob>((resolve) => {
      recorder.onstop = () => {
        teardownStream()
        resolve(new Blob(chunks, { type: mimeType }))
      }
      recorder.stop()
    })
  }

  function cancel(): void {
    recorder.onstop = null
    try {
      recorder.stop()
    } catch {
      // already stopped
    }
    teardownStream()
  }

  return { stream, stop, cancel }
}
```

```typescript
      const existing = draft[cellId]

      }
      draft[cellId] = clip

    setClips(draft => {
      const existing = draft[cellId]
      if (existing !== undefined) {
        disposeClip(existing)
        delete draft[cellId]
      }
    })

    setClips(draft => {

        disposeClip(draft[cellId])
        delete draft[cellId]
      }
    })
  }

  return {
    get clips() {
      return clips
    },
    setClip,
    clearClip,
    getClip,
    clearAll,
  }
}
```

```typescript
import { createSignal } from "solid-js"
import { audioContext, resumeAudio } from "../media/audio-context"
import type { Clip } from "./clip"

export type TransportState = "stopped" | "playing"

export type Transport = {
  state: () => TransportState
  /** Seconds since the current play pass began. 0 when stopped. */
  position: () => number
  /** Begin playback. Schedules audio at `audioContext.currentTime + 0.05`. */
  play: (clips: Clip[]) => Promise<void>
  stop: () => void
}

const SCHEDULE_LEAD_SECONDS = 0.05

export function createTransport(): Transport {
  const [state, setState] = createSignal<TransportState>("stopped")
  const [startedAt, setStartedAt] = createSignal(0)
  let sources: AudioBufferSourceNode[] = []
  let stopTimer = 0

  async function play(clips: Clip[]) {
    if (state() === "playing") {
      stop()
    }
    if (clips.length === 0) {
      return
    }
    await resumeAudio()
    const audio = audioContext()
    const when = audio.currentTime + SCHEDULE_LEAD_SECONDS
    sources = []
    for (const clip of clips) {
      const source = audio.createBufferSource()
      source.buffer = clip.audio
      source.connect(audio.destination)
      source.start(when)
      sources.push(source)
    }
    setStartedAt(when)
    setState("playing")

    const longest = Math.max(...clips.map((clip) => clip.duration))
    stopTimer = window.setTimeout(() => {
      stop()
    }, (longest + 0.1) * 1000)
  }

  function stop() {
    for (const source of sources) {
      try {
        source.stop()
      } catch {
        // not yet started
      }
    }
    sources = []
    window.clearTimeout(stopTimer)
    stopTimer = 0
    setState("stopped")
    setStartedAt(0)
  }

  function position() {
    if (state() !== "playing") {
      return 0
    }
    return audioContext().currentTime - startedAt()
  }

  return { state, position, play, stop }
}
```

```typescript
function render(
  viewport: ViewportState,
  leaves: LeafFrame[],
  frames?: ReadonlyMap<string, TextureSource>,
) {
  gl.clearColor(0, 0, 0, 0)
  gl.clear(gl.COLOR_BUFFER_BIT)

  if (leaves.length === 0) {
    return
  }

  // ---- Pass 1: color fill for every leaf (existing logic) ----
  const snap = Math.round
  ensureBufferSize(leaves.length)
  for (let index = 0; index < leaves.length; index++) {
    const leaf = leaves[index]
    const left = snap(leaf.rect.x + viewport.x)
    const top = snap(leaf.rect.y + viewport.y)
    const right = snap(leaf.rect.x + leaf.rect.width + viewport.x)
    const bottom = snap(leaf.rect.y + leaf.rect.height + viewport.y)
    positionBuffer[index * 2] = left
    positionBuffer[index * 2 + 1] = top
    sizeBuffer[index * 2] = right - left
    sizeBuffer[index * 2 + 1] = bottom - top
    const [r, g, b] = parseColor(leaf.color)
    colorBuffer[index * 3] = r
    colorBuffer[index * 3 + 1] = g
    colorBuffer[index * 3 + 2] = b
  }

  gl.useProgram(program)
  attributes.a_corner.bind()
  attributes.i_position.set(positionBuffer.subarray(0, leaves.length * 2)).bind()
  attributes.i_size.set(sizeBuffer.subarray(0, leaves.length * 2)).bind()
  attributes.i_color.set(colorBuffer.subarray(0, leaves.length * 3)).bind()
  uniforms.u_canvasSize.set(cssWidth, cssHeight)
  uniforms.u_view.set(0, 0, viewport.scale)
  gl.drawArraysInstanced(gl.TRIANGLE_STRIP, 0, 4, leaves.length)

  // ---- Pass 2: video frames for leaves that have one ----
  if (frames === undefined || frames.size === 0) {
    return
  }

  gl.useProgram(videoProgram)
  gl.activeTexture(gl.TEXTURE0)
  gl.bindTexture(gl.TEXTURE_2D, videoTexture)
  gl.uniform1i(videoSamplerLocation, 0)
  videoView.uniforms.u_canvasSize.set(cssWidth, cssHeight)
  videoView.uniforms.u_view.set(0, 0, viewport.scale)
  videoView.attributes.a_corner.bind()

  for (let index = 0; index < leaves.length; index++) {
    const leaf = leaves[index]
    const source = frames.get(leaf.id)
    if (source === undefined) {
      continue
    }
    // Reuse the same snapped position/size already computed above.
    videoPositionBuffer[0] = positionBuffer[index * 2]
    videoPositionBuffer[1] = positionBuffer[index * 2 + 1]
    videoSizeBuffer[0] = sizeBuffer[index * 2]
    videoSizeBuffer[1] = sizeBuffer[index * 2 + 1]
    videoView.attributes.i_position.set(videoPositionBuffer).bind()
    videoView.attributes.i_size.set(videoSizeBuffer).bind()

    gl.texImage2D(
      gl.TEXTURE_2D,
      0,
      gl.RGBA,
      gl.RGBA,
      gl.UNSIGNED_BYTE,
      source as TexImageSource,
    )
    gl.drawArraysInstanced(gl.TRIANGLE_STRIP, 0, 4, 1)
  }
}
```

```typescript
        }
      }
    }
    return frames
  })

function playbackTick() {
  const frames = gatherFrames()
  const drawViewport: ViewportState = { x: lastViewport.x, y: lastViewport.y, scale: 1 }
  if (frames.size > 0) {
    renderer.render(drawViewport, lastLeaves, frames)

      context.preview.activeCellId() !== null ||
      context.transport.state() === "playing",
```

```typescript
async function play(clips: Clip[], loopLength: number | null = null) {
  if (state() === "playing") {
    stop()
  }
  if (clips.length === 0) {
    return
  }
  await resumeAudio()
  const audio = audioContext()
  const firstWhen = audio.currentTime + SCHEDULE_LEAD_SECONDS
  scheduleSources(clips, firstWhen)
  setStartedAt(firstWhen)
  setState("playing")

  if (loopLength !== null) {
    function cycle() {
      if (state() !== "playing") {
        return
      }
      for (const source of sources) {
        try {
          source.stop()
        } catch {
          // not running
        }
      }
      sources = []
      const audio2 = audioContext()
      const nextWhen = audio2.currentTime + 0.01
      scheduleSources(clips, nextWhen)
      setStartedAt(nextWhen)
      stopTimer = window.setTimeout(cycle, loopLength * 1000)
    }
    stopTimer = window.setTimeout(cycle, loopLength * 1000)
  } else {
    const longest = Math.max(...clips.map((clip) => clip.duration))
    stopTimer = window.setTimeout(() => {
      stop()
    }, (longest + 0.1) * 1000)
  }
}

function scheduleSources(clips: Clip[], when: number) {
  const audio = audioContext()
  for (const clip of clips) {
    const source = audio.createBufferSource()
    source.buffer = clip.audio
    source.connect(audio.destination)
    source.start(when)
    sources.push(source)
  }
}
```

```typescript
function deleteSelection() {
  const selection = app.selection
  if (selection === null) {
    return
  }
  const targetedPath = selection.path.slice(0, selection.path.length - selection.depth)

  // Collect all entity ids in the subtree being removed.
  const removedIds: string[] = []
  function collectIds(node: Node) {
    if (node.type === "entity") {
      removedIds.push(node.id)
      return
    }
    for (const child of node.children) {
      collectIds(child)
    }
  }
  const targetNode = resolveNode(app.layout, targetedPath)
  if (targetNode !== null) {
    collectIds(targetNode)
  }

  const parentPath = targetedPath.slice(0, -1)
  const nextSelection: Selection = { path: parentPath, depth: 0 }

  setApp(app => {
    const next = removeAt(app.layout, targetedPath)
    app.layout = next ?? createEntity()
    app.selection = nextSelection
  })

    clips.clearClip(cellId)
  }
  if (Object.keys(clips.clips).length === 0) {

}
```

## Notes for the implementer

- **Style discipline.** Every memory rule (braced ifs, full names, typed window augments, `logAction`, `pnpm`, etc.) applies to all code you write. If you find yourself writing `if (!x) return`, fix it.
- **Mobile testing.** `getUserMedia` requires a secure context. To test from a phone on your LAN: `pnpm dev --host`, then enable `chrome://flags#unsafely-treat-insecure-origin-as-secure` and add `http://<lan-ip>:5173`. For real-world test, use a self-signed cert via `vite-plugin-mkcert`.
- **Video orientation.** Front-camera captures on Android may carry rotation metadata that `VideoDecoder` does NOT apply. If decoded frames render sideways, the fix is a rotation matrix in the renderer's video program (read the WebM metadata, expose rotation degrees on `Clip`, apply in the vertex shader). Spec doesn't require this for MVP; flag if you encounter it during smoke.
- **Performance.** Per-leaf `texImage2D` upload is fine for 4-9 cells. Beyond that, switch to a texture array. Cell upload order = `leaves` traversal order; if textures cause frame drops, instrument with `performance.measure` first.
- **Spec is authoritative.** When in doubt about scope, read `docs/superpowers/specs/2026-05-11-eddy-mvp-design.md`. Don't add features the spec doesn't include (export, persistence, tempo, etc.).
- **If a ported file resists pruning**, rewrite the relevant function from the WebCodecs / mediabunny / Web Audio docs rather than fighting the eddy-1.0 over-engineering. The goal is simple, not line-for-line faithful.
